# Página de Teste - Pixel Perfect Inspector

Esta é uma página HTML simples e bonita para testar a extensão Pixel Perfect Inspector.

## Como usar

1. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run test:page
   ```

2. **Ou use o comando dev padrão:**
   ```bash
   npm run dev
   ```
   E acesse: `http://localhost:3000/test/index.html`

3. **Carregue a extensão no Chrome:**
   - Vá para `chrome://extensions/`
   - Ative o "Modo do desenvolvedor"
   - Clique em "Carregar sem compactação"
   - Selecione a pasta `dist/` do projeto

4. **Teste a extensão:**
   - Abra a página de teste no navegador
   - Clique no ícone da extensão para ativar o modo de inspeção
   - Passe o mouse sobre os elementos para ver as propriedades CSS
   - Clique em dois elementos para medir a distância entre eles

## Elementos na página de teste

A página inclui vários elementos para testar:

- **Tipografia**: Headings, parágrafos, texto inline
- **Cards**: Containers com padding e border-radius
- **Botões**: Diferentes estilos de botões
- **Layout Flexbox**: Elementos com display flex
- **Imagens**: Placeholders para testar elementos de imagem
- **Formulários**: Inputs, textareas e labels
- **Espaçamentos**: Elementos com diferentes gaps para medir distâncias
